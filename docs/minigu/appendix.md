# 附录

## 附录一：MiniOB

MiniOB 是 [OceanBase](https://github.com/oceanbase/oceanbase) 团队基于华中科技大学数据库课程原型，联合多所高校重新开发的、专为零基础的同学设计的数据库入门学习项目。我们的目标是为在校学生、数据库从业者、爱好者或对基础技术感兴趣的人提供一个友好的数据库学习项目。

+ MiniOB的项目地址：[https://github.com/oceanbase/miniob](https://github.com/oceanbase/miniob)
+ MiniOB官网：[https://oceanbase.github.io/miniob/](https://oceanbase.github.io/miniob/)
