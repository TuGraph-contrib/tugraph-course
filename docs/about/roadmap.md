# Roadmap

以下是 TuGraph 图数据库课程的后续迭代计划：

- MiniGU 图数据库系列
    - 增加更多实验内容，涵盖高级查询与优化技术
    - 引入图算法实验，如最短路径、社区检测等
    - 提供更多实战项目，提升动手能力
- TuGraph-DB 系列: TuGraph-DB是工业级图数据库，后续将推出基于TuGraph-DB的系统化课程资源，涵盖环境搭建、API使用、图算法编写等内容
- Apache GeaFlow 系列: Apache GeaFlow 是开源的流图计算引擎，后续将推出基于 GeaFlow 的系统化课程资源，涵盖架构设计、DSL与API开发、实战案例等内容
- Graph+AI 技术系列：TuGraph社区有覆盖图智能体、Text2GQL、RAG等多方面的Graph+AI技术内容，后续将整理成系统化的课程资源
